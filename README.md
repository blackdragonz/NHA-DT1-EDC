## **บทที่ 1 หน้าหลัก**

### 1.1 ระบบแสดงหน้า Asset Management&#x20;

* ค้นหา
* บริหารจัดการ QR Code
* บริหารสัญญาจัดประโยชน์
* Dashboard มีรายละเอียดรายงาน ดังนี้
  * ข้อมูลกลุ่มสัญญาจัดประโยชน์ร่วม
